# W5 Moderator Run-Sheet

Use one row in [capture-table.md](capture-table.md) per participant.

## Before The Session

- Confirm Apple Silicon Mac and A2-B1 self-study fit.
- Confirm D+1 and D+7 availability.
- Start from a clean install: clear localStorage, IndexedDB, keys, cards, and local model state.
- Confirm the app launches without Gatekeeper workaround.
- Confirm Settings -> W5 validation is empty or reset.
- Have a stopwatch ready.

## Handover

1. Read [consent-script.md](consent-script.md).
2. Start at the first app screen.
3. Start the stopwatch at handover.
4. Stay silent until the participant completes the first task, asks for help, or gives up.

## Observe The Unaided Task

The target task is:

```text
curated native clip -> save one phrase -> review -> write one sentence -> correct -> save correction
```

Record timestamps for:

- first-run started;
- first clip played;
- first phrase saved;
- first review started;
- first review completed;
- mistake submitted;
- correction saved;
- first loop completed.

Also record:

- where the participant hesitates;
- the first step that blocks progress, if any;
- whether native audio is noticed unprompted;
- whether mistakes becoming drills is noticed unprompted;
- whether the participant attempts an own-source import.

If the participant asks for navigation help, mark unaided completion `N` before helping. A neutral
reminder to think aloud is allowed; instructions about where to go or what to click are not.

## Questions After The Task

Ask in this order:

1. "Me explica com suas palavras para que serve o PhraseLoop."
2. "O que voce percebeu de diferente em relacao ao que usa hoje?"
3. "Mostra como voce transforma ingles real em pratica hoje."
4. "Quando foi a ultima vez que voce salvou uma frase para estudar depois? O que foi chato?"
5. "Voce usaria o PhraseLoop no lugar desse processo pelos proximos 7 dias?"
6. "O que isso substituiria, se substituir alguma coisa?"
7. "O que faria voce desinstalar amanha?"
8. "Voce paga por algo hoje para aprender ingles?"
9. "Pelo que voce realmente pagaria R$19-39/mes: receber frases e audio prontos sem instalar
   ou configurar nada, revisar tambem no celular com o progresso atualizado, receber conteudo
   pronto para o seu nivel, nada, ou outra coisa?"

## After The Session

- Fill every capture-table column while memory is fresh.
- Copy TTFR and dropoff from Settings -> W5 validation when available.
- Log D+1 and D+7 follow-up due dates.
- Send the exact follow-up messages from [followup-messages.md](followup-messages.md).
